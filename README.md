# Controle-de-Estoque
Projeto desenvolvido em Java utilizando Programação Orientada a Objetos (POO).  O sistema permite o cadastro e gerenciamento de produtos perecíveis e não perecíveis, aplicando conceitos como herança, abstração, polimorfismo e sobrescrita de métodos.  ## Funcionalidades  - Cadastro de produtos - Controle de estoque - Listagem de produtos
